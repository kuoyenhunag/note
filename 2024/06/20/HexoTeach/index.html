如何使用Hexo在GitHub建置基礎的部落格

身為一個菜鳥工程師，在工作中時常會用到以前沒用過的技能，或是以前學過的東西常常就丟回去給老師了，所以開始想把自己整理的一些小筆記透過部落格的方式輸出出來，當作一個小小的紀錄。

所以開始研究怎麼使用Hexo透過github建立簡單的部落格，順便分享出來，給有需要的人參考看看。

這邊將略過如何安裝Node.Js以及Git的教學，需要教學的朋友，可以至下方參考文章觀看或自行搜尋喔～

安裝完成後可以到官方網站Hexo，選擇自己喜歡的主題配置即可。

建立完成後就可以進入你的VSCode終端機，並且在資料夾下輸入部署的指令：

接著是修改 _config.yml 設定檔中，有關 deploy 的設定（是根目錄的，並非為themes中的）。

之後輸入部署指令

接下來依照順序，會依序執行 clean -> generate -> deploy

後續需要到專案中的Setting -> Pages -> Branch ，修改你的Branch至master，這樣就完成部署成功了！
